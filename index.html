Frequently asked questions

Everything you need to know about AMOURA.

Yes. AMOURA is a bookmarklet — lightweight JavaScript that runs in your browser tab. There’s no installer, no download, and nothing saved to your computer. Questions are sent to our Cloudflare Worker for AI processing, but all traffic is AES encrypted end-to-end. The script also uses VM obfuscation — the strongest JavaScript protection available — so it can’t be read or tampered with.
No. Bookmarklets are sandboxed by your browser — they can only interact with the page you're currently on. They can't read other tabs, can't run in the background, and stop entirely when you navigate away.
Only the question text and answer choices are sent to the AI for processing — sometimes with a screenshot for visual questions. A random session ID is used for continuity. No personal information, login credentials, or cookies are ever collected or stored.

Yes. AMOURA works on mobile — iOS Safari and Firefox on Android both support bookmarklets. For the best experience, we recommend desktop Chrome or Edge.
AMOURA reads the question from the page, takes a screenshot if needed, and sends it to Gemini 3. The answer usually comes back in 1–3 seconds and gets entered automatically. Harder questions are routed to Gemini 3 Pro; simpler ones use Gemini 3 Flash. Pure arithmetic is solved locally without any API call.
AMOURA remembers wrong answers and won't pick them again. If it keeps missing, it automatically switches to a stronger model. Accuracy gets better the longer a session runs.
